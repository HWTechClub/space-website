The Aim

The Hermes Project aims to change the way rockets are built using modules of machinery made with different purposes such as boosters, fuel cells, sensors, landing systems and more. The overall objective is to create a module-based design for rockets which allows interchangeability within a class of equipment such as fuel cells where one out of multiple options can be used.

The First Stage

Alpha-H

The first prototype, called Alpha-H, will be a sugar motor rocket which will be used to gain an understanding of how to improve according to mechanical and software observations. The prototype will contain heat sensors, a parachute for a landing mechanism and an altimeter to judge the maximum height is reached.

The Second Stage

Based on the results of the first stage, the second stage will deal with modifying the prototype into modular parts once the mechanics have been understood. This modularization may not encompass all the systems in the second stage.
To test the modularised parts, another prototype, called Beta-H, will be designed, built, implemented and tested using them. This prototype will be the same model as the prototype in the first stage however it will be comprised of mostly only modular blocks.

The Third Stage

Omega-H

The third stage will focus on creating multiple modules based on the results of the testing phase of the second stage.
Multiple modules for fuel cells, boosters, sensors, and others will be created and tested by creating a few prototypes using different models than in the first and second stages. The number of prototypes will defer based on the efficiency of the modules but at least one prototype, called Gamma-H, will be created for this stage. This prototype will be built with mostly only modules like in the second stage but since it uses a different model, it will test the reliability and efficiency of the modular system.
